Waffle/Features

Native Platforms

Waffle supports, or has planned support, for the following platforms.

  • GLX
  • X11/EGL
  • Wayland
  • Android (planned)
  • MacOS/CGL (planned)
  • Windows (planned)

OpenGL API's

Waffle supports creating contexts from the API's below. Of course, The set of supported API's, and the available contexts that can be created for each, are dependent on the native system. For details on the relationship among native system, OpenGL API, and context creation, see the documentation for waffle_config.

  • OpenGL
  • OpenGL ES1
  • OpenGL ES2
