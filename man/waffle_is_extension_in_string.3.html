Name

waffle_is_extension_in_string — Check if a name appears in an OpenGL-style extension string

Synopsis

#include <waffle.h>
bool waffle_is_extension_in_string( const char *restrict extension_string,
  const char *restrict extension_name);
 

Description

waffle_is_extension_in_string()

Check if an extension name appears in an OpenGL-style extension string.

The extension_string must be in the format returned by glGetString(GL_EXTENSIONS). Note that this function is not restricted to the GL extension string; it can also be used on the GLX and EGL extension strings.

This function can be called before waffle has been successfully initialized with waffle_init(3).

Errors

This function sets the error code to WAFFLE_NO_ERROR.
